My First Assignment 01
📌 Project Overview

This is my first HTML and CSS project.

I created a simple Developer Conference Website to practice my web development skills.

In this project I learned how to use HTML for the website structure and CSS for the design.

🛠️ Technologies Used
HTML5
Used to create the structure of the website.
CSS3
Used to design the website and make it look beautiful.
📄 Website Sections
Hero Section
A welcome banner with a heading, short description and button.
Event Information
Shows information about the developer conference.
Schedule Section
Displays Day 1, Day 2 and Day 3 schedule.
Speakers Section
Introduces the conference speakers.
Pricing Section
Shows different ticket prices.
Contact Section
Contains contact information.
Footer
Displays the footer of the website.
